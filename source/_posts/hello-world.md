## Quick Start

### Create a new post

```bash
$ hexo new "My New Post"
```

More info: [Writing]()

### Run server

```bash
$ hexo server
```

More info: [Server]()

### Generate static files

```bash
$ hexo generate
```

More info: [Generating]()

### Deploy to remote sites

```bash
$ hexo deploy
```

More info: [Deployment]()
